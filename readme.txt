== Description ==

PopUp Video Generator is a simple to use plugin for wordpress that allows you to add a nice popup feature to your FLV videos. Simply insert a tag of code and the plugin will take care of the rest. 


== Example ==

[popupvideo "Title (required)" "Image Link (optional)" "FLV Link (required)" "PopUp Image Link (required)"]

[popupvideo "Test PopUp Video" "/wp-content/uploads/2010/09/example-image.jpg" "/wp-content/uploads/2010/09/video.flv" "/wp-content/uploads/2010/09/example-image.jpg"]

Note: If the Image Link field is left blank a text link will be generated with the title of the video.
